# La Salle Solidaria

## Project Description
La Salle Solidaria is an initiative aimed at promoting social welfare and support for the less fortunate within our community. This project allows users to contribute towards various charitable causes, facilitating donations and resources in an organized manner.

## Features
- **User Registration**: Users can create accounts to manage their donations and contributions.
- **Donation Tracker**: A feature to keep track of all donations made by users.
- **Charity Listings**: Display a list of charities and causes that users can contribute to.
- **Notification System**: Send alerts and updates to users about ongoing campaigns or new charities.

## Structure
The project is organized as follows:
- **/src**: Contains all source code files.
- **/data**: Holds JSON files for static data used in the application.
- **/tests**: Contains unit tests for the application.
- **README.md**: This documentation file providing insights into the project.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/rogercolls07/LaSalleSolidaria
   ```
2. Navigate into the project directory:
   ```bash
   cd LaSalleSolidaria
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Follow the instructions in the web app to create an account and start contributing!
